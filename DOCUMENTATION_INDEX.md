### Quick Problem Solving
| Problem | Solution |
|---------|----------|
| Can't add expense | Check internet, try manual entry |
| Voice input not working | Allow microphone, use manual entry |
| Can't find an expense | Use date filter, refresh page |
| Forgot password | Click "Forgot?" on login page |
| Friend didn't get group invite | Re-send invite code via WhatsApp |
| Bill split is wrong | Check if all members are selected |
| Can't see group expenses | Wait for page to load, refresh |
