## 🎯 Quick Navigation

| I want to... | How | Time |
|-------------|-----|------|
| **Add expense** | Dashboard → "Add Expense" OR click mic 🎤 | 15 sec |
| **Add via voice** | Click red mic 🎤, say "500 for food" | 10 sec |
| **Create group** | Dashboard → Groups → "Create Group" | 1 min |
| **Invite friend** | Group → "Share Invite Code" → Send code | instant |
| **Join group** | Dashboard → Groups → Paste code → Join | 30 sec |
| **Split bill** | Group → "Add Expense" → Select members | 1 min |
| **Plan trip** | Group → "Plan Trip" → Enter destination | 2 min |
| **Play roulette** | Group → "Bill Roulette" → Spin → Share | 1 min |
| **Track goal** | Dashboard → My Dreams → "Create Goal" | 2 min |
| **View settlement** | Group → "Settlements" section | instant |
| **Mark as paid** | Settlements → Click person → Mark Settled | 30 sec |

## 🎤 Voice Input Quick Words

### Say Amount First
✅ **Correct**: "500 for dinner"  
❌ **Wrong**: "dinner for 500"

### Category Keywords
```
🍕 Food: chai, dinner, lunch, pizza, khana, दूध
🚗 Travel: petrol, uber, bus, train, auto, यात्रा
🛍️ Shopping: kirana, amazon, mall, clothes, खरीदारी
📱 Bills: rent, electricity, wifi, recharge, बिल
📦 Other: anything else
```

### Example Phrases
- English: "500 for coffee" ☕
- Hindi: "500 का खाना" 🍛
- Hinglish: "250 ka chai" 🫖
- Hybrid: "Three fifty for uber" 🚕

## 👥 Group Management

### Adding Members
1. Create group
2. Enter member names
3. Click "Create"

### Removing Members
1. Go to group
2. Click X next to member name
3. Confirm removal

### Who Owes What
- **Green** ✅ = They owe you money
- **Red** ❌ = You owe them money
- **Gray** ⚪ = Already settled

## 🗺️ Trip Planning Quick Script

1. **Where**: "Jaipur" or "Goa" or "Kerala"
2. **How many**: "5 people" (or count members)
3. **Budget**: "10000" per person
4. Click **"✨ Generate Trip Plan"**
5. Wait 2 seconds
6. See places, hotels, food, tips
7. Click **"Share on WhatsApp"** 📱

## 🎡 Bill Roulette Play Guide

1. Click "Bill Roulette" button
2. Wheel appears with member names
3. Click "Spin"
4. Wheel spins for 3 seconds
5. Stops on a name (that person pays!)
6. Funny Bollywood quote appears 😂
7. Click "Share on WhatsApp"
8. Send to group
9. Laugh together! 🤣

## 🔒 Security Checklist

✅ **DO**:
- Use strong password (letters + numbers + symbols)
- Logout on public computers
- Enable microphone only when needed
- Keep app updated

❌ **DON'T**:
- Share password with anyone
- Use same password as other apps
- Post expense screenshots online
- Logout and forget to clear browser

## 🛠️ Troubleshooting Quick Fixes

| Problem | Fix |
|---------|-----|
| **App loads slow** | Refresh page, check internet |
| **Can't add expense** | Check internet, click "Add Expense" again |
| **Voice not working** | Allow microphone, speak clearly, restart |
| **Can't find expense** | Use date filter, scroll, refresh page |
| **Group visible but empty** | Wait to load, refresh, check filters |
| **Friend can't join group** | Resend invite code via WhatsApp |
| **Settlement math wrong** | Check all members are selected |
| **Old expenses missing** | Use custom date filter in Dashboard |
